## What's New

### Major Features

**Complete Client CRUD API**
- `GET /v1/clients/{id}` — Retrieve single client
- `PATCH /v1/clients/{id}` — Update client details and status
- `DELETE /v1/clients/{id}` — Archive client (soft delete)
- All endpoints include tenant-scoping, RBAC, and audit logging

**Production-Grade React UI**
- `ClientForm.jsx` — Reusable form component with validation
- `ClientModal.jsx` — Modal wrapper for create/edit workflows
- Enhanced Clients panel with add, edit, delete functionality
- Real-time list refresh after mutations
- Loading, error, and empty states

**Comprehensive Documentation**
- Updated OpenAPI v1.0.0 specification
- Release notes in `docs/RELEASE_1.0.0.md`
- Audit logging trails for all operations
- HIPAA-ready soft-delete pattern

## Upgrade Instructions

### Step 1: Update Code
```bash
git pull origin main
pnpm install
```

### Step 2: Build
```bash
cd apps/web && pnpm build
```

### Step 3: Restart Services
```bash
node ops/start-all.mjs
```

### Step 4: Verify
- Open http://localhost:3002/index.html
- Verify Clients panel shows "New Client" button
- Click New Client and verify form opens

**No data migration required.**
