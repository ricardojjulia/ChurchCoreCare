# v4.7.0 Release Summary

## Expanded Counseling Form Library

**Release Date:** March 29, 2026  
**Version:** 4.7.0  
**Status:** Production Ready  
**Scope:** Documents library expansion

## What Is Included

### Form Library Expansion

- Expanded the Documents module from 19 forms to 39 forms
- Added 20 new forms covering:
  - consent and administrative workflows
  - deeper clinical assessments
  - treatment-planning templates
  - therapeutic worksheets
  - faith-integrated reflection tools

### New Form Modules

- `apps/web/src/components/Documents/forms/AdministrativeForms.js`
- `apps/web/src/components/Documents/forms/ClinicalFoundationForms.js`
- `apps/web/src/components/Documents/forms/TreatmentPlanningForms.js`
- `apps/web/src/components/Documents/forms/TherapeuticWorksheets.js`
- `apps/web/src/components/Documents/forms/FaithCounselingForms.js`

### Catalog and Workflow Updates

- Expanded `apps/web/src/components/Documents/formRegistry.js`
- Expanded `apps/api/src/index.js` default form catalog
- Updated `apps/web/src/components/Documents/DocumentsPage.jsx` copy for the broader library scope
- Added a browser regression in `tests/e2e/high-value-journeys.spec.mjs` for opening a new consent form from the live Documents UI

### Documentation

- Added `PLANS/FORM-LIBRARY-EXPANSION.md`
- Updated `README.md`
- Updated `docs/change-log.md`
