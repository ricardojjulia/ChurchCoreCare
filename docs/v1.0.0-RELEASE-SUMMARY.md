## What Is Included

### API

- `GET /v1/clients`
- `POST /v1/clients`
- `GET /v1/clients/{id}`
- `PATCH /v1/clients/{id}`
- `DELETE /v1/clients/{id}` (soft-delete/archive)

### UI

- `ClientForm.jsx` for create and edit flows
- `ClientModal.jsx` for modal presentation
- Enhanced client panel actions in `WorkspaceGrid.jsx`
- Client list refresh integration in `App.jsx`
